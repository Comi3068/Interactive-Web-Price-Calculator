# Interactive Web Price Calculator

A simple interactive price calculator built with HTML, CSS, and JavaScript.  
Allows users to input values and see real-time calculations. Fully responsive.

## Live Demo
[View Live Site](https://interactive-web-price-calculator.vercel.app/)

## Technologies
- HTML
- CSS
- JavaScript
- Vercel (deployment)
